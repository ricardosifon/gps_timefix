gps_timefix

Fix TRACEBUF2 timestamps using local system time ‚Äì for Earthworm seismic systems

üîß About

gps_timefix is an Earthworm module that replaces invalid GPS-derived timestamps in TYPE_TRACEBUF2 messages with the local system time, helping stabilize acquisition systems during GPS failures.

üì¶ Features

  • Works with any TRACEBUF2 message
  • Non-invasive correction (starttime and endtime only)
  • Input/output via Earthworm rings

üöÄ Quick Start

git clone https://github.com/ricardosifon/gps_timefix.git
cd gps_timefix
make -f makefile.unix
./gps_timefix gps_timefix.d

üìò Configuration Example

# gps_timefix.d
MyModuleId        MOD_GPS_TIMEFIX
RingName          GPS_IN_RING
OutRingName       GPS_OUT_RING
HeartBeatInterval 30
LogFile           2
GetEventsFrom     INST_WILDCARD MOD_WILDCARD TYPE_TRACEBUF2
